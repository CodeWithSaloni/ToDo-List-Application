# To-Do List Application

## Overview

A console-based To-Do List Application developed using C#. The application allows users to add tasks, remove completed tasks, and view all pending tasks through a menu-driven interface.

## Features

* Add New Tasks
* Remove Existing Tasks
* View All Tasks
* Menu Driven Navigation
* Dynamic Task Storage using List<string>

## Technologies Used

* C#
* .NET
* Collections (List<T>)
* Loops
* Conditional Statements

## Learning Outcomes

* Worked with List Collections
* Implemented Menu Driven Programs
* Practiced CRUD-like Operations
* Improved Understanding of Console Applications
